# Keycloak IAM Training Course – Complete Syllabus (Beginner to Advanced)

## Keycloak Identity and Access Management (IAM) Training

**Target Audience:**  
Anyone interested in the Identity and Access Management (IAM) domain and Keycloak. Programming knowledge is required for some practical sessions.  

**Language:** English  

This course provides a strong practical foundation in Keycloak, covering concepts from IAM fundamentals to advanced integration, customization, and observability.

---

# Session 1 – Introduction & Environment Setup

## Topics

- Introduction to Identity and Access Management (IAM)
- Overview of Keycloak
- Comparison with other IAM products in the market
- Understanding Single Sign-On (SSO)
- OAuth 2.0 Overview
- OpenID Connect (OIDC)
- JSON Web Tokens (JWT)
- OAuth 2.0 Grant Types and when to use each

## Practical Sessions

- Install Keycloak using Docker
- Understanding multi-tenancy in Keycloak
- Realms and Organizations
- Hands-on with Keycloak Admin Console
- Connect Keycloak to PostgreSQL using Docker (with data persistence)
- Install and configure pgAdmin
- Create realms, users, roles, and groups
- Create an OIDC client for a backend application
- Secure a backend application using Keycloak authentication

---

# Session 2 – Applications, Authentication & Federation

## Topics & Practical Sessions

- Create an OIDC client for a Single Page Application (SPA)
- Secure SPA using Keycloak authentication
- Understanding browser cookies and authentication flow
- Enable Multi-Factor Authentication (MFA)
- Explore Keycloak database schema using pgAdmin
- User Federation with LDAP
- Login using LDAP user credentials

---

# Session 3 – Customization & Deployment

## Topics & Practical Sessions

- Customize Keycloak authentication flows
- Develop and integrate a custom Service Provider Interface (SPI)
- Install and configure NGINX as a load balancer
- Configure custom domain for Keycloak
- Host multiple Keycloak instances behind NGINX
- Review Keycloak clustering and jdbc_ping sessions

---

# Session 4 – Advanced Integration & Observability

## Topics & Practical Sessions

- Overview of SAML protocol
- Identity Provider (IDP) Brokering
- Integrate external Identity Providers
- Configure social login using OIDC

## Observability

- Enable Keycloak observability features
- Run Keycloak on Minikube
- Configure Prometheus scraping
- Monitor metrics for production readiness

---

# Session 5 – Keycloak Configuration & Automation

## Realm Import and Export

- Export existing realm configuration
- Import realm into another environment
- Backup and replication strategies

## Configuration as Code using Terraform

- Manage Keycloak configuration declaratively
- Version control IAM configuration
- Automate environment setup
- Build reproducible infrastructure

---

# Course Outcome

By the end of this training, participants will:

- Understand IAM fundamentals
- Implement OAuth2 and OpenID Connect
- Deploy production-ready Keycloak environments
- Integrate applications securely
- Implement MFA and federation
- Configure observability
- Automate Keycloak using Terraform

---

# Pre-requisites

- Docker must be installed on the participant’s laptop or PC.
- Basic understanding of web applications.
- Programming knowledge recommended for SPI development sessions.
